# Disable Right Click Plugin for WordPress

This lightweight WordPress plugin protects your content by disabling right-click, copy-paste, text selection, and access to developer tools.

## Features
- Disable right-click
- Disable copy, paste, cut, and drag
- Block Dev Tools (F12, Ctrl+Shift+I)
- Optional alert messages
- Clean, lightweight code

## Installation
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin from your WordPress dashboard
3. Optional: Configure settings via the admin panel (if available)

## Plugin Folder
- `disable-right-click-plugin.php` – Main plugin file
- `protection.js` – Frontend JavaScript to block actions
- `settings-page.php` – Admin panel (optional)
